## 第一段：实机演示（3分钟）— 镜头对准 Dobot + 桌面

### 0:00-0:20 开场

**镜头**: Dobot 全景 + 桌面场景（目标物体、障碍物、放置区清晰可见）

"各位老师好。我们小组的项目是『世界模型前瞻规划：推-抓组合任务』。"

"现在看到的是真实 Dobot 机械臂和桌面场景。桌上有一个红色目标物体、一个蓝色障碍物，以及绿色标记的放置区。当前是 Layout 2——障碍物挡在目标前方，机械臂无法直接抓取。"

（用手指一下各个物体）

---

### 0:20-0:50 系统启动 + 感知

**镜头**: 切到电脑屏幕，展示终端运行命令

"我们在终端启动系统："

```
python scripts/run_task.py --layout 2 --mode dobot
```

**镜头**: 切回 Dobot

"系统首先通过相机感知场景，检测到目标物体位于坐标 (230, 0) 毫米，障碍物位于 (180, 0) 毫米，两者距离约 50 毫米。障碍物正好挡住了抓取路径。"

---

### 0:50-1:40 世界模型规划 + 推动作

**镜头**: Dobot 侧面，能看清末端和物体

"由于无法直接抓取，系统调用世界模型进行前瞻规划。"

"世界模型在后台模拟了 200 条候选推动作——从不同角度、不同距离去推障碍物——预测每条动作执行后物体最终会到哪。评分函数根据物体离放置区的距离和是否仍被遮挡来打分。"

"规划器选择了最优动作：从障碍物前方 5 毫米处，以 90 度方向向右推约 60 毫米。"

**镜头**: 紧盯 Dobot 末端，不要动

"现在执行推动作。"

（Dobot 末端移动到起点 → 下降 → 直线推 → 抬起）

"障碍物被成功推开。系统重新感知——确认目标物体已经暴露，可抓取性从 false 变为 true。"

---

### 1:40-2:20 抓取 + 放置

**镜头**: Dobot 全景

"接下来执行抓取。机械臂移动到目标正上方，下降，夹爪闭合，抬起。"

（Dobot 执行抓取动作）

"然后移动到放置区上方——绿色标记位置——下降，夹爪松开。"

（Dobot 执行放置动作）

"任务完成。从感知到规划到执行，全过程自主完成。"

### 2:20-2:50 另两种布局快速展示

**镜头**: 快速切换场景

"我们设计了三种初始布局。刚才演示的是 Layout 2——最典型的推-抓组合场景。"

（快速展示 Layout 1 和 Layout 3 的运行，每种 10-15 秒）

"Layout 1 是无障碍基线场景，机械臂直接抓取。Layout 3 是物体在角落，需要长距离推近后才能抓。三种布局在我们的批量实验中均达到 100% 成功率。"

## 第二段：屏幕讲解（3分钟）— 展示架构和可视化

### 3:00-3:30 系统架构

**屏幕**: 打开系统架构图

"这是我们的系统架构。四个核心模块构成闭环："

"感知模块从相机获取 RGB-D 图像，检测物体位置；世界模型接收当前状态和候选动作，预测下一状态；规划器在候选动作空间中搜索最优序列；机械臂控制模块执行选中的动作。执行后重新感知，形成滚动重规划。"

---

### 3:30-4:30 世界模型

**屏幕**: 依次展示三张图

"这是我们训练的世界模型的表现。"

**第一张: loss_curves.png**
"模型在 DCU 服务器上训练了 200 个 epoch，使用 5000+ 条仿真交互数据。蓝色是训练集损失，橙色是验证集损失。可以看到两者都在稳定下降，最终验证集 MSE 收敛到约 23 mm²。"

**第二张: world_model_validation.png**
"这是预测精度验证图。横轴是真实物体位移，纵轴是模型预测的位移。每个点代表一次推动作——如果完美预测，点应该落在对角线上。我们的 MLP 模型在 X 和 Y 方向上的 RMSE 分别为 4.80mm 和约 5mm。"

**第三张: ablation_success_rate.png**
"消融实验对比了三种世界模型：朴素启发式模型、MLP 模型和 GRU 模型。在三组场景各 10 次试验中，训练后的 MLP 模型达到 100% 成功率，而启发式模型在复杂场景下成功率有所下降。"

---

### 4:30-5:20 前瞻规划

**屏幕**: 展示规划器可视化或代码逻辑

"规划器采用 CEM——交叉熵方法。在每步决策时，采样 200 条候选动作，用世界模型预测每条动作的效果，选出评分最高的精英子集，迭代优化动作分布。"

"这是滚动重规划的示意：绿色虚线是世界模型『想象』的轨迹，红色实线是实际执行后的轨迹。可以看到，世界模型的预测和真实物理基本一致，规划器因此能做出有效决策。"

---

### 5:20-5:50 总结

**屏幕**: 结果汇总

"总结一下："

"我们实现了一个完整的推-抓组合系统——感知模块从 RGB-D 检测物体，世界模型预测动作效果，CEM 规划器选择最优推，Dobot 机械臂执行推-抓-放置。"

"三种布局各 20 次试验，共 60 次，成功率 100%。"

"我的演示完毕，谢谢各位老师。"
