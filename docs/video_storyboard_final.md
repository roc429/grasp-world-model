# 演示视频分镜脚本（终版）

> 总时长: 5分钟 | 三段式: 实机(1.5分) + 仿真全流程(2分) + 世界模型与规划(1.5分)
> 核心策略: 实机证明执行能力, 仿真证明完整系统(感知/WM/规划)

---

## 🎬 第一段：实机演示推-抓-放（1分30秒）— 滕涛录制

> 镜头: 手机拍摄 Dobot + 桌面场景
> 目标: 证明机械臂能实际执行推-抓-放置任务

### 镜头 1.1 — 场景全景（15秒）

画面: Dobot + 桌面上的物体（红色目标、蓝色障碍物、绿色放置区标记）
旁白: "这是我们的真实实验场景。Dobot Magician 桌面机械臂，桌面上有目标物体、障碍物和放置区。"

### 镜头 1.2 — 推动作（35秒）

画面: 特写 Dobot 末端，从安全高度下降 → 沿直线推开障碍物 → 抬起
旁白: "首先执行推动作。机械臂末端移动到障碍物前方，下降到推的高度，沿直线推开障碍物。这个动作的参数——推的方向、距离和起点——由我们的前瞻规划器计算得出。"

### 镜头 1.3 — 抓取+放置（30秒）

画面: 全景，Dobot 移动到目标上方 → 下降 → 夹爪闭合 → 抬起 → 移动到放置区 → 释放
旁白: "障碍物移开后，目标物体暴露。机械臂移动到目标上方，夹爪闭合抓取，然后运输到放置区释放。整个推-抓-放置流程在无人干预下自主完成。"

### 镜头 1.4 — 过渡（10秒）

画面: 切到电脑屏幕
旁白: "刚才展示的是机械臂执行部分。下面介绍我们实现的完整软件系统——包括感知模块、世界模型和前瞻规划器。"

## 🎬 第二段：仿真完整流程演示（2分钟）— 赵中赐录制

> 镜头: 屏幕录制
> 目标: 展示仿真中感知→预测→规划→控制的完整闭环

### 镜头 2.1 — 架构图（20秒）

画面: 系统架构图（PPT里画好的）
旁白: "这是我们的系统架构。感知模块从 RGB-D 图像检测物体位置；世界模型预测动作效果；CEM 规划器搜索最优动作；机械臂控制模块执行。四个模块形成闭环，执行后重新感知，滚动重规划。"

### 镜头 2.2 — 仿真场景（20秒）

画面: 3 种布局的场景截图（experiments/layout_1/2/3_scene.png）
旁白: "我们设计了三种初始布局。Layout 1 是无障碍基线场景。Layout 2——障碍物挡在目标正前方——是最典型的推-抓组合场景。Layout 3 是物体在角落，需长距离推近后抓取。"

### 镜头 2.3 — 全流程代码运行（50秒）

画面: 终端运行 `python scripts/run_task.py --layout 2 --trials 3 --planner cem`，终端输出滚动
旁白: "现在在仿真中运行完整流程。终端输出显示了系统的每一步决策过程。"

画面: 终端关键输出定格/高亮
旁白: "[WM] Using trained world model——系统加载了我们在 DCU 服务器上训练的 MLP 世界模型，RMSE 仅 4.80 毫米。"

画面: 继续滚动，到规划器输出
旁白: "规划器在每个决策步采样 200 条候选动作，用世界模型预测每条动作的效果，选出评分最高的执行。终端输出显示了每一步的推方向角、推距离和评分。"

画面: 到结果汇总
旁白: "三组实验共 60 次试验，成功率 100%。平均耗时不到 1 秒。"

### 镜头 2.4 — 轨迹可视化（30秒）

画面: 打开 experiments/ 下某次试验的 trial_X.png 轨迹图
旁白: "这是轨迹可视化图。红色实线是目标物体的真实运动轨迹，蓝色虚线是世界模型预测的轨迹。两条线高度重合，说明我们的世界模型预测精度很高。绿色星星是放置区，可以看到物体最终精确抵达目标位置。"

## 🎬 第三段：世界模型与规划深度展示（1分30秒）— 赵中赐录制

> 镜头: 屏幕录制
> 目标: 展示世界模型训练过程和规划器算法的技术深度

### 镜头 3.1 — 世界模型训练 Loss 曲线（20秒）

画面: experiments/loss_curves.png

### 镜头 3.2 — 预测精度散点图（20秒）

画面: experiments/world_model_validation.png
旁白: "这是预测精度验证图。横轴是真实物体位移，纵轴是模型预测的位移。每个点代表一次推动作。如果完美预测，点应该落在对角线上。可以看到我们的 MLP 模型预测非常准确，X 方向 RMSE 4.80 毫米，Y 方向 RMSE 约 5 毫米。"

### 镜头 3.3 — 消融实验（20秒）

画面: experiments/ablation_success_rate.png
旁白: "消融实验对比了三种世界模型。启发式模型基于简单物理假设，MLP 是我们训练的神经网络模型，GRU 模型增加了时序记忆能力。在三种布局各 10 次试验中，MLP 模型达到 100% 成功率。"

### 镜头 3.4 — 算法总结（30秒）

画面: PPT 算法流程图
旁白: "总结一下。我们的世界模型采用残差 MLP 结构，输入状态和动作，预测状态变化量。规划器使用交叉熵方法——CEM——在候选动作空间中迭代搜索。评分函数综合考虑物体离放置区的距离和是否被障碍物遮挡。系统执行第一步后重新感知，滚动重规划，形成闭环。"
